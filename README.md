# Blackish — The Clothing

Official website for **Blackish**, an India-based street culture clothing brand.

Built with pure HTML, CSS & JavaScript. No frameworks, no dependencies.

## 🔗 Live Site
[blackish-website.netlify.app](#) ← update this after deploying

## 📸 Instagram
[@blackish_theclothing](https://www.instagram.com/blackish_theclothing)

## 🛠 Tech
- HTML5 / CSS3 / Vanilla JS
- Google Fonts (Bebas Neue, Syne, DM Sans)
- Scroll reveal animations
- Custom cursor
- Fully responsive

## 📦 How to Run
Just open `index.html` in any browser. That's it.
